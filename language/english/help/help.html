Help: Tad Lunch3

Description

This module is for accessing campus food ingredients login platform to pick up designated school lunch information

Install/Uninstall

No special requirements, follow the standard installation procedure to install the module – unzip the module, upload the module to the modules directory, and install it from the “Module” in the “Module” management interface in the background. If you need more detailed installation instructions, please see the XOOPS Operations Manual (English).
